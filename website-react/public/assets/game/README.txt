九型小游戏结果页 GIF 放置说明
=============================
把对应人物的 GIF 放到本目录，按九型编号命名：

  result-1.gif   完美型
  result-2.gif   助人型
  result-3.gif   成就型
  result-4.gif   自我型
  result-5.gif   观察型
  result-6.gif   忠诚型
  result-7.gif   活跃型
  result-8.gif   领袖型
  result-9.gif   和平型

- 建议正方形（如 512x512），会被裁成圆形显示。
- 未放置时，结果页自动显示「GIF 占位 · 待替换」框，不影响布局。
- 替换后无需改代码，刷新即可生效。
